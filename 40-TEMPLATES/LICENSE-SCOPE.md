# Template License Scope

License for template assets in this directory tree: `UNLICENSE` (public-domain style).

Covered files:

- `40-TEMPLATES/TEMPLATE-*.md`
- `40-TEMPLATES/*.template`
- `40-TEMPLATES/INSTRUCTION-STARTER-PACK/*.template*`
- `40-TEMPLATES/INSTRUCTION-STARTER-PACK/AGENTS.template.*.md`

If a file includes its own license notice, that file-level notice takes precedence.

Non-template code in other folders should use that folder's own license policy
(for example, `LAB-scheduling-demo/LICENSE` for service code).
